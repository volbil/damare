{# Cover renderer. Two modes: • If `url` is given (and truthy), render an with that source. • Otherwise, render a deterministic SVG via cover_palette()/cover_layout(). The SVG path is the original behaviour and serves as a fallback when a novel has no real cover image (originals, missing data, etc.). #} {% macro cover(title, author='', seed='', show_text=true, url=None) -%} {%- if url -%} {%- else -%} {%- set palette_seed = seed if seed else title -%} {%- set pal = cover_palette(palette_seed) -%} {%- set lay = cover_layout(seed, title) -%} {%- set bg = pal[0] -%} {%- set accent = pal[1] -%} {%- set ink = pal[2] -%} {%- endif -%} {%- endmacro %}
